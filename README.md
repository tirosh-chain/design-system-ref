# Design System Reference

Spectrum, Carbon, Mantine을 같은 의료 업무 시나리오 기준으로 비교하는 레퍼런스 앱입니다.

## Live Demo

https://tirosh-chain.github.io/design-system-ref/

## Scenarios

- Patient Intake Form: 입력, 선택, 검증, 동의 상태 비교
- Monitoring Data Table: 상태 요약, 진행률, 환자 리스트 비교
- Alert Workflow: 위험 알림, 단계 진행, 조치 액션 비교
